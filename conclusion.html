Conclusions:

  • The recommendation system using item-item collaborative filtering and popularity-based approaches, along with GNN for similar item recommendations, has shown promising results in improving the accuracy and relevance of product recommendations.
  • The precision and recall scores obtained through evaluations indicate that the system is able to effectively recommend relevant products to the users.
  • The implementation of GNN has shown potential in improving the accuracy and relevance of similar item recommendations.

Future Work:

  • Integration of additional recommendation techniques, such as content-based filtering, to improve the overall accuracy and relevance of recommendations.
  • Exploring different graph neural network architectures and techniques to improve the accuracy and relevance of similar item recommendations.
  • Incorporating user feedback and preferences to personalize the recommendations for individual users.
  • Scaling up the recommendation system to handle larger datasets and user traffic.
  • Conducting more extensive evaluations and comparing the performance of the system with other state-of-the-art recommendation systems.
